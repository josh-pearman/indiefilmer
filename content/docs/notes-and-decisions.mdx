## Production Notes

The notes section is a free-form space for production notes. Organize notes by category — pre-production, on-set, post-production, or any custom category you create. Each note has a title, content, and optional attachments.

Use notes for:

- Meeting notes from production meetings
- Shot lists and storyboard references
- Creative direction documents
- Vendor quotes and correspondence

## Attachments & Links

Attach files and add links to any note. Upload reference images, PDF contracts, audio files, or link to external resources like Google Docs or Dropbox folders.
