## Daily Meal Plans

Each shoot day gets its own meal plan. Configure up to three meals:

- **Crafty** — snacks and drinks available throughout the day
- **Lunch** — main meal, usually after 6 hours
- **Dinner** — for longer shoot days

For each meal, set a vendor, estimated cost, and actual cost. You can also add notes (e.g. "Vegetarian options needed").

## Automatic Headcounts

Headcounts are calculated automatically from the cast and crew assigned to each shoot day, with crew assignments tracked per day. No more manually counting heads — the system knows exactly who's on set each day.

## Dietary Restrictions

Dietary restrictions from cast and crew profiles are displayed in a dedicated dialog (access via "View Dietary" link). The dialog groups restrictions by type and shows individual names under each, so you can quickly see all dietary needs at a glance when planning meals.

## Standalone Craft Services Days

Create meal entries for non-shoot days — rehearsals, table reads, wrap parties, location scouts, or any event needing catering. Add a standalone day with a date, label, location, and headcount. You can also mark dates as TBD if you don't know the exact date yet.

When you have unlinked craft services days, you can link them to shoot days so they inherit cast, crew, and dietary data from that shoot day.

## Empty State

If no craft services entries exist, you can start by creating a standalone craft services day directly or by adding shoot days in the Schedule page.
