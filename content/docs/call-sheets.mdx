## Generating a Call Sheet

Call sheets are generated from shoot days. Navigate to a shoot day and click "Generate Call Sheet." The system pulls together all the data for that day into a professional, printable format.

## What's Included

Each call sheet includes:

- **Header** — project name, shoot day number, date
- **General call time** — when crew should arrive
- **Weather** — forecast for the shoot day
- **Sunrise/Sunset** — critical for exterior shoots
- **Location** — address with an embedded map
- **Crew list** — all assigned crew with individual call times
- **Cast list** — assigned cast with personal call times and pickup info
- **Scene breakdown** — scenes scheduled for the day with details
- **Announcements** — important notes for the day

## Printing & Export

Call sheets are designed to look great when printed. Use your browser's print function (Ctrl/Cmd + P) to print or save as PDF. The print layout automatically hides navigation elements and optimizes the layout for paper.
