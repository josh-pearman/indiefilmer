## Script Versions

The Script Hub manages all versions of your screenplay. Upload script files and tag them with version names following industry conventions:

- White — original draft
- Blue — first revision
- Pink — second revision
- Yellow, Green, Goldenrod — subsequent revisions

Or use your own naming convention — "Draft 1", "Final Draft", "Locked Script", etc.

## Current Version

Mark one version as "Current" to indicate which script everyone should be working from. The current version is highlighted and shown prominently in the Script Hub.

## Revision Tracking

Track page counts across versions and add notes to each revision explaining what changed. This creates a paper trail of script evolution — useful for tracking revisions that affect scheduling, cast, or budget.
