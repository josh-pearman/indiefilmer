## Overview

indieFilmer is a production planner built for microbudget and indie filmmakers. It brings your schedule, scenes, cast, crew, locations, catering, gear, budget, tasks, and notes into one place — so you can stop juggling spreadsheets and focus on making your film.

Everything is organized around a single **project**. Each project represents one production and contains all of its data: shoot days, scenes, cast members, crew, locations, and more.

## Creating a Project

When you first log in, you'll be prompted to create a project. Give it a name (e.g. "My Short Film") and you're in. You can always create additional projects later from the project switcher.

Once inside a project, you'll see the sidebar with links to every section. The sidebar is your primary navigation — everything is one click away.

## Navigation

The sidebar contains links to all major sections:

- **Dashboard** — overview of your project status
- **Schedule** — plan shoot days
- **Scenes** — manage your scene breakdown
- **Cast & Roles** — track actors and roles
- **Crew** — manage your crew roster
- **Locations** — scout and manage locations
- **Catering** — plan meals for each shoot day
- **Gear** — build equipment packages
- **Budget** — track planned vs. actual spending
- **Tasks** — a kanban-style task board
- **Notes** — production notes and decision log
- **Script Hub** — manage script versions

On mobile, the sidebar collapses into a hamburger menu accessible from the top-left corner.

## Project Settings

From the sidebar, click the gear icon to access project settings. Here you can:

- Rename your project
- Set production dates
- Configure project-level defaults
- Manage team members and their access roles
